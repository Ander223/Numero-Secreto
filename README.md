# Numero-Secreto
Este é um projeto de estudo para aprender a utilização do gitHub, um curso da Alura que contem aulas e exercicios para 
praticar e fixar o aprendizado.
Este é um projeto no qual o programa gera um número e depois pede para o usuário digitar um número para tentar adivinhar
e assim segue até  que o mesmo acerte o número secreto.
